Time of Day

A JavaScript Project by Matdeli Parra

A web page that uses JavaScript to determine the current hour, and presents a different image, custom message, and color based on each of the following times of day:

Morning (5am to 12 noon)
Afternoon (12 noon to 6pm)
Evening (6pm to 11pm)
Night (11pm to 5am)
